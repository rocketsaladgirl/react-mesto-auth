# "Место (MESTO)" - проект страницы в социальной сети для Яндекс.Практикум.

## *React - первое знакомство!*

В этом спринте мы по прежнему продолжаем улучшать проект "MESTO".

Мы продолжаем улучшать проект с помощью библиотеки REACT.
В данной работе мы добавили возможность добавлять, удалять, лайкать и дизлайкать карточки, а так же поработали с элементами popup, вынеся их в отдельные компоненты.

Работа обрела почти готовый вид, но есть еще элементы (например валидация), над которыми нужно поработать.

Готовый вариант можно посмотреть по ссылке: https://rocketsaladgirl.github.io/mesto-react/

Функционал будет расширяться на следующих этапах обучения.
